# Amazon_sales_analysis
Data analytics project showcasing amazon sales analysis using python, SQL and Power BI
End-to-end data analysis project on 120,350 Amazon orders using Python, PostgreSQL, and Power BI.

## Tools
- **Python** — data cleaning, feature engineering, EDA (pandas, NumPy, Matplotlib)
- **PostgreSQL** — data storage and SQL aggregation views
- **Power BI** — interactive dashboard with KPIs, charts, and slicers
